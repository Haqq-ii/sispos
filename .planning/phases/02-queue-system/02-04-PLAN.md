- ProtectedRoute allowedRoles check is defense-in-depth; the server enforces roles authoritatively.

Purpose: Plans 02-05, 02-06, 02-07 all import from the files created here. No page code yet — just infrastructure.

Output: ProtectedRoute extended, useAntrianStore, 3 hook files, jadwal FE Zod schema.
