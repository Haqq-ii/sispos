    Internal function getJadwalTersedia(wargaId: string, tanggal?: string): Promise<object>:
    - Load warga: prisma.warga.findUnique({ where: { id: wargaId }, select: { posyanduUtamaId: true } })
    - If !warga?.posyanduUtamaId: return { error: 'Citizen belum memilih posyandu utama.' }
    - dateFrom = tanggal ? new Date(tanggal) : new Date(); dateTo = new Date(dateFrom); dateTo.setDate(dateTo.getDate() + 7)
    - Query: prisma.jadwal.findMany({ where: { posyanduId: warga.posyanduUtamaId, statusJadwal: 'aktif', tanggalPelaksanaan: { gte: dateFrom, lte: dateTo } }, include: { slotSesi: { select: { id: true, labelSesi: true, jamMulai: true, kuota: true, terisi: true } }, posyandu: { select: { namaPosyandu: true } } }, orderBy: { tanggalPelaksanaan: 'asc' } })
    - Map to clean object: { id, namaPosyandu, tanggalPelaksanaan, sesi: [{ id, labelSesi, tersedia: kuota-terisi }] }
    - Return result

    Internal function getProfilBalita(wargaId: string): Promise<object>:
    - prisma.balita.findMany({ where: { wargaId }, select: { id: true, namaBalita: true, tanggalLahir: true, jenisKelamin: true } })
    - Return array (empty array if no balita)

    Internal async function executeToolCall(toolName: string, args: Record<string, string>, wargaId: string): Promise<string>:
    - switch(toolName):
        case 'get_jadwal_tersedia': return JSON.stringify(await getJadwalTersedia(wargaId, args.tanggal))
        case 'get_profil_balita': return JSON.stringify(await getProfilBalita(wargaId))
        case 'daftar_antrian': result = await ambilAntrian(args.slotId, args.balitaId, wargaId); return JSON.stringify({ success: true, antrianId: result.antrianId, nomorUrut: result.nomorUrut, estimasiMenit: result.estimasiMenit })
        case 'batalkan_antrian': result = await batalkanAntrian(args.antrianId, wargaId); return JSON.stringify({ success: true, ...result })
        case 'reschedule_antrian':
            old = await prisma.antrian.findFirst({ where: { id: args.antrianId, wargaId } })  // IDOR guard
            if (!old) return JSON.stringify({ error: 'Antrian tidak ditemukan atau bukan milik Anda.' })
            await batalkanAntrian(args.antrianId, wargaId)
            newResult = await ambilAntrian(args.slotId, old.balitaId, wargaId)
            return JSON.stringify({ success: true, oldAntrianId: args.antrianId, newAntrianId: newResult.antrianId, nomorUrut: newResult.nomorUrut })
        default: return JSON.stringify({ error: 'Unknown tool' })
    - Wrap each case in try/catch; on error return JSON.stringify({ error: e.message ?? 'Tool execution failed' })

    Export async function chatPendaftaran(wargaId: string, userMessage: string, clientHistory: Array<{ role: string; content: string }>): Promise<{ reply: string; messages: Array<{ role: string; content: string }> }>:
    - Graceful degradation: if !process.env.OPENAI_API_KEY → return { reply: 'Asisten pendaftaran tidak tersedia saat ini.', messages: [...clientHistory, { role: 'user', content: userMessage }, { role: 'assistant', content: 'Asisten pendaftaran tidak tersedia saat ini.' }] }
    - const OpenAI = (await import('openai')).default; const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY })
    - Build messages array (OpenAI.Chat.ChatCompletionMessageParam[]):
        [{ role: 'system', content: PENDAFTARAN_SYSTEM_PROMPT }, ...(clientHistory as OpenAI.Chat.ChatCompletionMessageParam[]), { role: 'user', content: userMessage }]
    - let iteration = 0; const MAX_ITERATIONS = 5
    - while (iteration < MAX_ITERATIONS):
        response = await openai.chat.completions.create({ model: 'gpt-4o', temperature: 0.4, messages, tools: TOOLS, parallel_tool_calls: false })
        choice = response.choices[0]
        messages.push(choice.message as OpenAI.Chat.ChatCompletionMessageParam)
        if choice.finish_reason === 'stop':
            reply = choice.message.content ?? 'Maaf, tidak bisa memproses permintaan.'
            // Persist only user + assistant content turns (not tool messages — VarChar(10) role constraint)
            await prisma.riwayatChat.createMany({ data: [{ wargaId, role: 'user', pesan: userMessage }, { wargaId, role: 'assistant', pesan: reply }] })
            // Return messages WITHOUT system prompt (client doesn't need it; strip tool messages too for clean client state)
            const clientMessages = [...clientHistory, { role: 'user', content: userMessage }, { role: 'assistant', content: reply }]
            return { reply, messages: clientMessages }
        if choice.finish_reason === 'tool_calls' and choice.message.tool_calls:
            for each toolCall of choice.message.tool_calls:
                args = JSON.parse(toolCall.function.arguments) as Record<string, string>
                result = await executeToolCall(toolCall.function.name, args, wargaId)
                messages.push({ role: 'tool', tool_call_id: toolCall.id, content: result } as OpenAI.Chat.ChatCompletionMessageParam)
        iteration++
    - After loop: throw Object.assign(new Error('AI tidak bisa menyelesaikan permintaan.'), { code: 'AI_TIMEOUT' })

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → Express API | wargaId from JWT only; clientHistory sent from browser is UNTRUSTED for wargaId purposes |
| Tool execution layer | All tool executions pass wargaId from JWT to antrian.service which has IDOR guards |
| System prompt | PENDAFTARAN_SYSTEM_PROMPT hardcoded server-side; client cannot override it |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04-01 | Tampering | Confirmation gate bypass via prompt injection | mitigate | parallel_tool_calls:false prevents parallel tool execution; PENDAFTARAN_SYSTEM_PROMPT explicitly forbids calling action tools before confirmation; both enforcement layers required |
| T-04-04-02 | Spoofing | Citizen A cancels Antrian of Citizen B via chatbot | mitigate | executeToolCall passes wargaId from JWT to batalkanAntrian and ambilAntrian; antrian.service.batalkanAntrian uses findFirst({ where: { id, wargaId } }) — IDOR guard already implemented |
