## Interaction Contract

### Month Picker

- Native `input[type=month]` — same widget as in `PuskesmasDashboardPage`.
- Default value: current month WIB, computed via `Date.now() + 7h` offset (matches dashboard pattern).
- Position: top-right of green header (same layout as PuskesmasDashboardPage).
- Style: `px-3 py-1.5 text-xs border border-[rgba(255,255,255,0.3)] bg-[rgba(255,255,255,0.15)] text-white rounded-[14px] focus:outline-none focus:bg-[rgba(255,255,255,0.25)]` — exact copy from `PuskesmasDashboardPage.tsx` line 100.
- On change: update local `bulan` state (no server call — download is on-demand).

### Stat Summary Boxes (optional but recommended for UX)

- Show 4 stat boxes in header after month picker — same 2×2 grid as PuskesmasDashboardPage.
- Source: `GET /api/dashboard/stats?bulan=YYYY-MM` (already exists from Phase 4 — reuse).
- Loading state: 4 skeleton boxes with `animate-pulse`.
- If no data: show `0` for all counts — no error shown for zero stats.

### Download Buttons

- Rendered as list rows inside the Format Laporan card (matching the stub's visual style).
- Each row: `FileSpreadsheet` or `FileText` icon (16px, `text-[#008236]`) + label + description + inline "Unduh" button.
- Click: `window.open(url, '_blank', 'noopener,noreferrer')` — called synchronously from onClick (not after await, to avoid popup blocking per RESEARCH.md Pitfall 6).
- URL pattern: `/api/reports/laporan-bulanan?bulan=${bulan}&format=xlsx` or `&format=pdf`.
- Disabled state: never disabled (bulan always has a default value; download proceeds even if month has no data — backend returns empty file).
- No loading spinner on download buttons — `window.open` is fire-and-forget.

### Loading States

| State | Component | Behavior |
|-------|-----------|---------|
| Stats loading | 4 shimmer boxes | `animate-pulse` same as PuskesmasDashboardPage |
| Stats error | Silently show `0` counts | Do not block download functionality |
| Download triggered | None | `window.open` is instant; no spinner needed |

### Warning / Info Notice

- Show always: a bottom card (same amber style as stub) with disclaimer text about e-PPGBM column format being academic-standard.
- Style: `bg-[#fffbeb] border border-[#fef3c6] rounded-2xl p-4` — exact match to existing stub.
- Prefix icon: `AlertCircle` 14px `text-[#973c00]`.

## States Summary

| State | Trigger | Visual |
|-------|---------|--------|
| Default | Page load | Month = current WIB; stat boxes loading; download rows enabled |
| Stats loaded | TanStack Query resolves | 4 stat boxes show real numbers |
| Stats zero | Month has no pemeriksaan | Stat boxes show 0; download still enabled (empty file) |
| Stats error | Network error | Stat boxes show 0; no toast (silently degrade) |
| Download clicked | User clicks Unduh | `window.open` fires; no UI change |
| Month changed | User changes input | Stat boxes refetch; download URL updates |

## Constraints (from CLAUDE.md + RESEARCH.md)

- Export via ExcelJS + pdfkit only — puppeteer is explicitly forbidden.
- `catatanKonsultasi` and `rekomendasiAi` MUST NOT appear in any export — UU PDP No. 27/2022.
- Download via `window.open` synchronously from onClick (not after await).
- Month picker uses native HTML `input[type=month]` — no date picker library.
- All UI text in Bahasa Indonesia.
- Date format in labels: `[nama bulan] [tahun]` using `toLocaleDateString('id-ID', { month: 'long', year: 'numeric' })`.
- Stats data uses `GET /api/dashboard/stats?bulan=YYYY-MM` (Phase 4 endpoint) — no new endpoint for display.
- Download endpoint: `GET /api/reports/laporan-bulanan?bulan=YYYY-MM&format=xlsx|pdf` (Phase 5 new endpoint).
