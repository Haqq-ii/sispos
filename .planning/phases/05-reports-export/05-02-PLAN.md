| Boundary | Description |
|----------|-------------|
| HTTP request → route handler | bulan and format come from req.query — must validate before use |
| JWT → puskesmasId | puskesmasId extracted from authMiddleware-verified JWT; never from req.query |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-01 | Elevation of Privilege | laporanBulananHandler — puskesmasId source | mitigate | `const puskesmasId = req.user!.userId` — line verified by grep acceptance criterion; req.query.puskesmasId never used |
